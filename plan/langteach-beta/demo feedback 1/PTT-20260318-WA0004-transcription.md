## Key Ideas Summary

1. **CEFR Curriculum alignment** - The product should incorporate the Common European Framework of Reference for Languages (A1, A2, B1, B2, etc.) to know what content students need at each level.
2. **Placement tests** - Ability to generate level assessment tests for new students.
3. **Lesson plan generation** - Given a target level/certification (e.g. B2), number of sessions, and hours, generate a structured learning plan covering grammar and communicative skills.
4. **Per-session activity planning** - For each session, suggest specific activity types (not generic ones).
5. **Student profile personalization** - Account for class composition (e.g. 3 Italians, 1 Chinese, 1 Iraqi, 1 German) to personalize activities based on L1 interference patterns and needs.
6. **In-person classes too** - Not just online, also consider presential teaching scenarios.
7. **Easy class tracking/follow-up** - After class, teacher can send a quick audio note describing how it went, and the system uses that to update and adapt future lesson plans.
8. **Evaluation tools** - Text correction that categorizes errors by type (grammar, lexicon, punctuation, verbal forms, etc.), going beyond what ChatGPT does generically.
