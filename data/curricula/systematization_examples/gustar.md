# Gustar -- Grammar Systematization Worksheet

**Source:** Iberia Language Academy
**File:** `4.Gustar.pdf`
**Type:** Student worksheet (reading + grammar exercises + conjugation table + opinion scale)
**Approximate CEFR level:** A1 (verb gustar, communicative use of likes/dislikes)
**Pages:** 4

## Format

Multi-page worksheet combining a reading text, comprehension and grammar exercises, a conjugation reference table, and a communicative opinion scale.

## Page-by-Page Breakdown

### Page 1 -- Reading Text

**Title:** "Lo que me gusta y lo que no"
A short first-person narrative text in which a speaker describes their tastes and preferences. Covers food, leisure activities, and daily life topics. Uses "gustar" and related verbs (encantar, molestar, odiar) in context.

### Pages 2-3 -- Exercises

- **Comprehension questions:** True/false or short-answer questions about the reading text
- **Grammar extraction:** Students identify and underline uses of "gustar" and note the subject in each sentence
- **Gap-fill / transformation exercises:** Students complete sentences using the correct form of gustar (gusta / gustan) based on singular vs. plural noun complement
- **Production exercise:** Students write 4-6 sentences describing their own tastes using the structures from the text

### Page 4 -- Reference Materials

**Conjugation table for GUSTAR:**

| Pronombre | Forma |
|-----------|-------|
| (a mi) | me gusta / me gustan |
| (a ti) | te gusta / te gustan |
| (a el/ella/usted) | le gusta / le gustan |
| (a nosotros) | nos gusta / nos gustan |
| (a vosotros) | os gusta / os gustan |
| (a ellos/ustedes) | les gusta / les gustan |

**Opinion scale (Likert-style, 5 points):**

| Expresion | Meaning |
|-----------|---------|
| Me encanta | I love it |
| Me gusta mucho | I like it a lot |
| Me gusta | I like it |
| No me gusta | I don't like it |
| Odio | I hate it |

## Pedagogical Purpose

Systematizes the indirect-object construction of "gustar" which is a known difficulty for learners (reversal of subject/object roles vs. English "to like"). The worksheet:
- Grounds the structure in comprehensible input (reading text)
- Provides guided analysis exercises before free production
- Gives students a portable conjugation reference
- Introduces the affective register through the opinion scale (encantar through odiar)
