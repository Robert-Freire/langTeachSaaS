# La Casa -- Vocabulary Systematization Card

**Source:** Iberia Language Academy
**File:** `la_casa.pdf`
**Type:** Visual vocabulary reference card (image-based labeling)
**Approximate CEFR level:** A1-A2 (household vocabulary)

## Format

Single-page color illustration of a cross-section house showing multiple rooms. Each room contains furniture and objects labeled with Spanish vocabulary terms. Labels use numbered callouts pointing to specific items in the image.

## Rooms Depicted

- Salon / Sala de estar (living room)
- Cocina (kitchen)
- Dormitorio (bedroom)
- Bano (bathroom)
- Terraza / Balcon (terrace/balcony)

## Pedagogical Purpose

Provides visual anchoring for household lexicon. Students match labels to image callouts. Typically used as:
- Introduction activity before a housing/accommodation unit
- Self-study vocabulary reference
- Basis for oral description practice ("En mi casa hay...")

## Notes

- No exercises included; pure reference card
- Complements units on "Tipos de vivienda" (A2 vocabulary theme in Iberia curriculum)
- Format is standard visual glossary, not a worksheet
