## Content Quality Issues

### [CQ-1] SYSTEM-WIDE: WarmUp always generates a vocabulary drill
**Severity:** High
**Affects:** All 5 personas (100% reproduction rate)
**Description:** Every WarmUp section contains a vocabulary block (word / definition / example sentence). Jordi has explicitly said: "warm-up is never vocabulary, it's more like break the ice, talk a bit, lose fear." A vocabulary drill is Presentation-level content and has no place in WarmUp.
**Example (Marco B1):** WarmUp = 8 vocab items (ubicado/a, de buen/mal humor, el estadio...) instead of a "Where's your favorite sports venue and why?" conversation prompt.
**Proposed fix:** Update the generation prompt for WarmUp to specify: conversational/discussion prompt, 2-5 minutes, no right/wrong answers, NOT a vocabulary list or grammar drill.

### [CQ-2] Ana Exam: C1 vocabulary in B2 lesson
**Severity:** Medium
**Affects:** Ana Exam Prep only
**Description:** "soslayar" (to sidestep/evade) in the WarmUp vocabulary block is C1+ level, not B2. For exam prep this is especially harmful — students study the wrong level.
**Proposed fix:** Add vocabulary level validation to the generation prompt or post-generation filter.

### [CQ-3] Ana Exam: vocabulary definitions in Spanish for English L1 student
**Severity:** Medium
**Affects:** Ana Exam Prep only
**Description:** WarmUp vocabulary definitions are in Spanish ("evitar un problema o dificultad sin resolverlo directamente") for a student whose L1 is English. Rubric requires translations.
**Proposed fix:** Prompt must specify: "provide English translations/definitions for students whose L1 is English."

### [CQ-4] Ana Exam: role-play Production in DELE exam prep lesson
**Severity:** Medium
**Affects:** Ana Exam Prep only
**Description:** The Production section is an oral role-play (travel agency conversation). DELE B2 production tasks are written (opinion essay, formal letter, short report). The wrong exercise type signals to the teacher that this is regular conversation practice, not exam preparation.
**Proposed fix:** Exam Prep template prompt should specify written production tasks with formal register.
